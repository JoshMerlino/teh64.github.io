Intro

Buttons are the easiest Material to master. They support most of the attributes and can be manipulated with ease. Let's see some others.

Adding a splash of color. (added alpha)

Coloring Buttons and almost all material is easy for the Material Basic colors use the color attribute. The color attribute only supports material colors. But this dosent mean your limited to only the 17 material and 10 holo colors, If you want to use other colors such as an rgb code, hex value or a css color your best bet is the buttons background-color attribute. If you do use the background-color css attribute its a good idea to also change the color of the button because it wont automaticly pick a color based on the background for you. If theres no color it will be a default grey.



<button color=".teal">Sample 1</button><button style="background-color:#80F">Sample 2</button>

Borders (added v1.0)

Each button has a default 2px thick bottom border that is 10% darker than the button. This can be removed with the md-border attribute. This accepts a true or false the default is true so its a waste of time putting md-border="true".



<button color=".cyan">With Border</button><button color=".cyan" md-border="false">Without Border</button>

Flat Buttons (added v2.0)

Flat buttons are the button that looks like a link or text but when you hover over it, it changes shape and morphs into a button. Well you can do this to with the md-flat attribute. this attribute accepts
no params if its there the system will treat it like a flat button.
