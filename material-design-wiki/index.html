Intro

Buttons are the easiest Material to master. They support most of the attributes and can be manipulated with ease. Let's see some others.

Adding a splash of color.

Coloring Buttons and almost all material is easy for the Material Basic colors use the color attribute. The color attribute only supports material colors. But this dosent mean your limited to only the 17 material and 10 holo colors, If you want to use other colors such as an rgb code, hex value or a css color your best bet is the buttons background-color attribute. If you do use the background-color css attribute its a good idea to also change the color of the button because it wont automaticly pick a color based on the background for you. If theres no color it will be a default grey.
